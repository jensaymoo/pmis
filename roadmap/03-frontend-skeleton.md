# Фаза 3 — Каркас фронтенда (пустые страницы)

> Все маршруты существуют, все страницы отдают заглушки. Навигация работает, данных нет.

> Фаза 3 не зависит от фазы 2 — ей нужны только результаты фазы 1. Фазы 2 и 3 можно выполнять параллельно.

## Что делаем

- `router/index.js` — все маршруты: `/login`, `/gantt`, `/work-orders`, `/works`, `/users`, `/resources/:kind`
- Пустые page-компоненты с заголовком:
  - `pages/LoginPage.vue` — «Вход в систему»
  - `pages/GanttPage.vue` — «Диаграмма Ганта»
  - `pages/WorkOrdersPage.vue` — «Наряды»
  - `pages/WorksPage.vue` — «Работы»
  - `pages/UsersPage.vue` — «Пользователи»
  - `pages/ResourcesPage.vue` — «Справочники ресурсов»
- `layouts/MainLayout.vue` — `n-config-provider` (русская локаль, светлая тема), хедер-заглушка, `<router-view>`
- `App.vue` — `<router-view>` (LoginPage вне лэйаута, остальные внутри)

## Готово когда

- Ручной ввод URL → рендерится соответствующая заглушка
- `/login` — без лэйаута; остальные — внутри MainLayout
- Переход между страницами без ошибок
